# AI 像素桌面宠物

一个完全离线、零 API 调用、纯 Python 实现的桌面 AI 宠物游戏。宠物会自主学习你的互动习惯，拥有独立的性格和记忆，并通过微型神经网络与你对话。

---

## 快速开始

### 环境要求

- Python 3.14+
- 依赖：`pygame-ce`, `PyQt5`, `numpy`

### 安装与运行

```bash
pip install pygame-ce PyQt5 numpy
cd ai_pet
python main.py
```

或双击 `run.bat` 启动。

首次启动会自动训练 Char-RNN 微型神经网络（约 30 秒），训练完成后弹出宠物窗口。

## 操作方式

| 操作 | 效果 |
|------|------|
| **左键单击** 宠物 | 随机互动，宠物切换对应表情 |
| **左键长按拖拽** | 移动宠物位置 |
| **右键单击** 宠物 | 弹出操作菜单 |
| **托盘左键单击** | 窗口置顶 |
| **托盘左键双击** | 窗口置顶 + 互动 |
| **托盘右键** | 操作菜单 |

### 右键菜单功能

- **打招呼** — 开心/好奇回应
- **摸摸头** — 提升心情，显示爱心特效
- **称赞** — 提升心情，显示星星特效
- **喂食** — 降低饥饿度，显示食物特效
- **送礼物** — 送礼物给宠物，大幅提升心情
- **洗澡** — 提升心情+健康，显示泡泡特效
- **治疗** — 恢复健康，显示闪光特效
- **玩耍** — 消耗精力提升心情，显示玩具特效
- **跳舞** — 消耗精力大幅提升心情，跳舞动画
- **讲故事** — 提升心情、消耗精力，显示书本特效
- **教本领** — 提升心情、消耗精力、增加好奇心
- **生气** — 降低心情，显示愤怒特效
- **改名字** — 自定义宠物名称
- **查看状态** — 弹出窗口显示属性进度条
- **退出** — 保存数据后退出

## AI 大脑架构

### 整体设计

```
用户互动 → 性格系统 → 行为引擎 → 动作输出
                  ↘ 记忆系统 ↗
对话 ← 混合引擎（模板 + Char-RNN）
```

### 四个子系统

#### 1. 行为引擎（Q-Learning）

基于 Q-Learning 的决策系统，宠物学习在什么状态下应该做什么。

- **状态空间**：饥饿 × 精力 × 心情 × 健康 × 时间段（5 个维度各 3 级离散）
- **动作空间**：`sleep`, `play`, `explore`, `cuddle`, `eat`, `wander`, `groom`
- **学习机制**：每个动作获得 reward，Q 表记录状态-动作价值
- **探索率**：20% 随机探索 vs 80% 利用经验

状态随 tick（1 秒）自动衰减，使用 δt 确保帧率稳定：
- 饥饿每秒 +0.02/60
- 精力每秒 -0.015/60
- 心情每秒 -0.01/60
- 健康每秒 -0.005/60（饥饿 > 80 或精力 < 20 时翻倍）

#### 2. 性格系统（5 维向量）

```
{
    "affectionate": 0.6,  亲昵度
    "energetic":    0.5,  活跃度
    "curious":      0.7,  好奇心
    "stubborn":     0.3,  固执度
    "clingy":       0.4,  粘人度
}
```

- 性格影响 Q-Learning 的 reward 权重（活泼型爱玩，亲昵型爱撒娇）
- 性格影响对话模板选择（亲昵的宠物说话更甜）
- 性格随互动缓慢变化（+0.001/次，每日总变化限 ±0.1）
- 性格变化规则：
  - 抚摸 → 亲昵↑，粘人↑
  - 喂食 → 亲昵↑
  - 玩耍 → 活跃↑，好奇↑
  - 责骂 → 亲昵↓，固执↑

#### 3. 记忆系统（时间衰减）

基于键值存储的记忆系统，所有记忆按 7 天半衰期衰减。

```
记忆示例：
    "last_fed":      {"time": 123456, "value": "鸡肉", "count": 5}
    "last_petted":   {"time": 123460, "count": 23}
    "favorite_spot": {"time": 123000, "value": "键盘旁边"}
```

- 权重公式：`weight = e^{-days / 7}`
- 权重低于 0.05 且记数 < 3 的记忆自动清理
- 记忆影响对话内容：*"你上次喂我鸡肉好好吃~"*

#### 4. 混合对话引擎

```
用户互动 → 确定情绪标签
        ├─ 80% 概率：模板匹配（保证通顺）
        │     "我很开心！{name}你真好~"
        │
        └─ 20% 概率：RNN 自由生成（增加惊喜）
              "<happy>我好想" → RNN 续写成 "我好想和你一起玩！"
```

模板系统覆盖 10 种情绪（happy/sad/hungry/sleepy/playful/curious/grateful/loving/excited/cuddly），每种情绪 8 条中文模板。

## 微型 Char-RNN

### 模型规格

| 参数 | 值 |
|------|------|
| 架构 | Embedding → 单层 RNN → Linear |
| 词表大小 | 70 字符（字母 + 数字 + 标点） |
| Embedding 维度 | 16 |
| 隐藏层维度 | 64 |
| 参数量 | ~28K |
| 权重文件 | ~112KB |
| 依赖 | 仅 numpy |

### 训练

- 语料库：中英文混合短句，按情绪标签分组（`<happy>`, `<sad>` 等）
- 训练方式：纯 NumPy 手动实现 RNN 反向传播（BPTT）
- 每个句子用 `<start>` / `<end>` 包裹，标签控制生成风格
- 首次启动自动训练 200 个 epoch（约 30 秒），学习率线性衰减
- 训练 loss 从 ~3.1 降至 ~0.59

## 矢量精灵

宠物以 128×128 的矢量风格卡通形象呈现，使用 Pygame 图元（椭圆、圆、多边形、弧线、抗锯齿线）组合绘制。

### 情绪动画状态（11 种）

| 状态 | 帧数 | 关键动画 | 实现方式 |
|------|------|----------|----------|
| idle | 2 帧 | 呼吸起伏 | sin 波驱动 body 垂直微动 |
| happy | 3 帧 | 上下弹跳 | bounce_y = abs(sin(frame*0.5))*6 |
| sad | 2 帧 | 垂头丧气 | head_tilt 负值 + 半闭眼 |
| playful | 3 帧 | 活泼好动 | 高弹跳 + 尾巴竖起 + 踢腿 |
| hungry | 2 帧 | 左右张望 | head_tilt 正弦摆动 + 张嘴 |
| sleepy | 2 帧 | 打瞌睡 | eye_open=0.1 + 缓慢点头 |
| sick | 2 帧 | 生病萎靡 | body_color 换为灰白色 |
| curious | 2 帧 | 好奇歪头 | head_tilt 摆动 + 睁大眼 |
| excited | 3 帧 | 兴奋蹦跳 | 高弹跳 + 举双爪 + 快速踢腿 |
| cuddly | 2 帧 | 撒娇求抱 | 倾斜歪头 + 伸爪 + 腮红 |
| dance | 3 帧 | 左右摇摆 | 交替抬爪 + 左右 sway |

### 互动视觉特效（9 种）

每个互动操作会触发对应的叠加特效，持续约 2-3 秒：

| 特效 | 触发操作 | 效果 |
|------|----------|------|
| 爱心 | 摸摸头 | 粉色心形上升飘散 |
| 食物 | 喂食 | 小鱼干从上方掉下 |
| 礼物 | 送礼物 | 礼物盒带蝴蝶结弹跳 |
| 玩具 | 玩耍 | 毛线球滚动 |
| 星星 | 称赞/跳舞 | 金色星星旋转上升 |
| 泡泡 | 洗澡 | 蓝色泡泡飘散 |
| 书本 | 讲故事 | 打开的书本 |
| 愤怒 | 生气 | 红色爆炸标记抖动 |
| 闪光 | 治疗/教本领 | 菱形闪光旋转 |

精灵完全由代码定义（Pygame 图元 + SRCALPHA 合成），无需任何外部图片资源。

## 桌面集成

### 窗口特性

- **透明背景**（`WA_TranslucentBackground`）
- **置顶显示**（`WindowStaysOnTopHint`）
- **无边框无标题栏**（`FramelessWindowHint`）
- **无任务栏条目**（Windows API `WS_EX_TOOLWINDOW`）
- **不窃取焦点**（`WA_ShowWithoutActivating`）

### 系统托盘

- 像素猫图标（代码绘制）
- 右键菜单包含所有互动功能
- 启动时弹出通知气泡

## 配置文件

`config.py` 中包含所有可调参数：

- 初始属性值（饥饿/精力/心情/健康）
- 属性衰减速率
- Q-Learning 超参数（学习率 / 折扣因子 / 探索率）
- RNN 架构参数（词表 / 维度 / 温度）
- 对话模板库
- 记忆半衰期
- 性格更新速率
- 互动效果参数
